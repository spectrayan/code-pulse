# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within CodePulse, please send an e-mail to **support@spectrayan.com**. All security vulnerabilities will be promptly addressed.

Please include the following information in your report:

* Type of issue (e.g., buffer overflow, SQL injection, cross-site scripting)
* Components affected
* Steps to reproduce
* Potential impact

We ask that you do not disclose the vulnerability publicly until we have had a chance to address it.

## Supported Versions

Currently, we only provide security updates for the latest stable release.

## Attribution Requirement

Please remember that as per our [LICENSE](./LICENSE), any modifications or updates made to CodePulse must include credit to **Spectrayan**.
