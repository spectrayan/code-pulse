# Airbnb JavaScript Style Guide (Summary)

Key conventions from the Airbnb JavaScript style guide.

## Variables
Use `const` by default, `let` when reassignment is needed. Never use `var`.

## Functions
Use arrow functions for anonymous functions. Use named function expressions over declarations.

## Objects & Arrays
Use object shorthand and computed property names. Use array spreads to copy arrays.

## Destructuring
Use object and array destructuring for accessing properties and elements.

## Strings
Use template literals for string interpolation. Prefer single quotes for strings.

## Naming
- `camelCase` for variables and functions
- `PascalCase` for classes and constructors
- `UPPER_SNAKE_CASE` for constants

## Modules
Use ES6 `import`/`export` over non-standard module systems. Do not use wildcard imports.
